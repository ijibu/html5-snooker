html5-snooker
==========

html5联网斯洛克游戏，初试代码是在websnooker.com上爬下来的，
然后自己处于兴趣，拿来分析修改，提供后台代码，最终实现联网的功能。

项目计划
	第一阶段
		后台用PHP支持
	第二阶段
		后台用go或者erlang重写
